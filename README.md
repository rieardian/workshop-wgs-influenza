# 🧬 Workshop WGS Influenza

**Indonesia | 2026**

---

## 📌 Overview

Workshop ini bertujuan untuk meningkatkan kompetensi petugas laboratorium dalam memahami dan melakukan analisis Whole Genome Sequencing (WGS) Influenza untuk mendukung surveilans genomik penyakit infeksi khususnya virus influenza

## 🧪 Output Workshop

Peserta diharapkan mampu:
1. Menjelaskan konsep dasar teknologi sekuensing genom
2. Memahami perbedaan metode partial sequencing dan Whole Genome Sequencing (WGS)
3. Menjelaskan alur kerja sekuensing influenza menggunakan berbagai platform sekuensing
4. Memahami proses analisis data genom menggunakan pipeline bioinformatika
5. Melakukan analisis mutasi dan variasi genom virus influenza
6. Mengunduh dan mengelola data genom dari database global
7. Melakukan editing dan pembersihan data sekuens
8. Membuat analisis phylogenetic tree
9. Memahami proses pelaporan dan pengunggahan data genom ke database global.
